About the Project

The first part of the project is an easy 3-D print of a nametag with my name on it. The point was to learn the basis of the software and how to adjust the project to print out the things that we want from it. An easy intro to printing 3-D objects.By reading the Prusa Mini Handbook, we learned how to prepare the print surface, calibration, filament, and work thorugh any issues that came with the print.

The second aspect of it was a personal print that tested the limitations that a 3-D print can have. I chose to print a stress test of a hollowed spiral to test how well the print can do with a smaller more delicate design that is meant for bigger adjustments. This is to notice and learn at how delicate 3-D printing really is, and how we need to change what we need to make it work better.

Overall, we see that the printer has amazing capability to create awesome projects, if given the right commands to do so. To push the limits of what a 3-D printer can for designing and printing is based on a lot of different variables. From the filament, temperatures, printing settings, software settings, patterns, there are so many vairables to learn and use in order to create better adjustments for future prints. For me, in specific prints, retraction settings, as well as adding supports can decrease the smaller mistakes caused by the temperature of the filament. By giving it a better support, and more time to cool, there would be less collapse and more structure to future sprints similar to this.

Name Tag Print

Learned to work the Prusa Slicer in order to create, adjust, and change, the aspect of creating an existing file for a print. Initially, adjusted to just print our names.

Working the Printer

So this is how the printing process works! It's all behind downloading some pre-made models, and we’re just adjusting it to our names in this case. When it first came to navigating the software, it was a tad bit confusing how all the different modes work (beginner, medium, expert) when it came to adjusting specific settings. For the most part it was all straightforward, and playing with the software was pretty fun. I had to ask my peers exactly how it all worked in terms of changing it to make sure I was using the right filament, to double checking if the machine actually took in the filament all the way, to adjusting the code to make sure it went from STL to 3MF. Little changes like this made the printing process actually work, so it's interesting to see how little adjustments make huge changes. Lastly, the difference between 3MF and STL is that 3MF has the complete model information contained for the printing job(modeling, colors, tectures, mesh,etc) while the STL only contains the mesh that needs to be printed. 3MF is better for designing and adding to the project, whereas STL is better for the simple mesh that needs to be printed.

Finished Product

The finished product! It made sense how the little details weren't perfect, there are only so many little details you can input for the size of the tag, so making size adjustments and things like that should make the tag/project look alot better. Can't wait till see how other projects react to such changes. Really enjoy the changes of colors that I chose, and wonder at how different depths of the tag change the whole outlook, or if changes to the borders would make that much of a difference. Eitherway, the tag looked great and I had no trouble with the printer or software so I was very happy with the outcome!

Stress Test

I was looking for an interesting print that would not only show the limitations of how the machine can function, but to also show how we need to consider the stability of the machine building, and filament as well. My specific torture test is a hollowed spiral with holes in between to show how delicate the material is to work with is, and how support is/isn’t necessary for prints. For this torture test, I was hoping ot see the limits of what the machine itself can do without changing setting for its initial run, seeing what important changes we need to take into account for future prints.

Failed Project

Halfway through the print, I noticed that it just fell apart, so I had to immediately stop the print to look over what happened. The base of the print looked well, until we began the actual structure of the spiral. From the very bottom, there is a light layer of foundation that the print went off, and was very thin that there was no way it could have held more given that the print was too hot so it melted into itself. The lack of suport had alot to do with it, but I do think that if the print was a tad bit slower or even bigger, it could have been succesful. Foundations without any support are either a hit or miss depeding on the infill, but for this case, it was definitely a fail. Additionally, the specific hollowed points of the spiral drooped into one another, and because the filament was too hot and not cooled off completely, then it fell into itself and messed up the print as it went on. Possibly if it was a bit slower in printing and gave it more time to cool, the structure wouldn't have fallen into itself. It finally fell over towards the end with the imbalance of the weight, bad structure, and that there was basically no more structured procedure to follow. The print was a successful failure, as it showed under-extrusion from the print, lots of stringing, and the melting itself. I was hoping to see more stringing from the center,but hopefully with resizing the print, the results would change.

Limits

Some of the limits for the machine are definitely the cooling times of the filament when it comes to smaller projects, and also the accuracy of where things go. I think that if it was more accurate when it came out the release of the filament, then the structure would have been much more firm. Additionally, the project without support would have ultimately failed, either structurally or visually, since there is nothing there to support it to be as accurate as possible, but regardless adjustments of such things can help make it more successful. Trial and error make it all better, and I do believe that with changes in the size and supporting system of the printing, the outcome would be alot better.
